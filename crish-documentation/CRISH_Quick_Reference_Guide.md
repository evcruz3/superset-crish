| Disease | Moderate | High | Severe |
|---------|----------|------|--------|

| Diarrhea | ≥25 cases | ≥50 cases | ≥100 cases |

- **Endemic**: <10
- **Alert**: 10-50

## 🔧 Common Tasks

### Export Data
1. Click on chart/table
2. Select "Download" ⬇️
3. Choose format:
   - CSV (data)
   - PNG (image)
   - PDF (report)

### Filter Data
1. Date range selector 📅
2. Municipality dropdown 📍
3. Disease type selector 🦠
4. Apply filters ✓

### Generate Report
1. Reports → New Report
2. Select template
3. Set parameters
4. Preview
5. Export/Schedule

## 🆘 Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| Can't login | Check CAPS LOCK, verify username |
| Dashboard slow | Clear cache, use Chrome/Firefox |
| Missing data | Check date range, refresh page |
| Export fails | Try smaller date range |
| Sync error | Check internet, retry |
